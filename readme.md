## 简单的九宫格的抽奖

### 基本逻辑
1. 点击抽奖后立马开始动画效果
2. 请求接口或者手工设置中奖结果
3. 减慢动画效果, 命中奖品


### 演示
[演示地址](https://xiangwenhu.github.io/lottery/index.html)   
1. 点击立即抽奖
2. 点击设置奖项
    奖项用逗号分隔, 最大值为7, 这些位置应该对应的是同一奖项.
